# aws_mqtt_bridge_broker
A bridge between AWS Device MQTT and a standard MQTT broker.
a) Folder structure: 
- package.json
- app.js
- node_modules
- certs

b) Copy your certs (received from Amazon AWS) in certs folder
c) RUN npm install
d) Edit the app.js script with your data
e) RUN npm start
f) Enjoy
